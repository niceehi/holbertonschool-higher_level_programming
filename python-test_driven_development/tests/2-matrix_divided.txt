matrix_divided tests
====================

>>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal case
>>> matrix_divided([[2, 4], [6, 8]], 2)
[[1.0, 2.0], [3.0, 4.0]]

Float case
>>> matrix_divided([[1.5, 3.0]], 1.5)
[[1.0, 2.0]]

Negative numbers
>>> matrix_divided([[-2, -4]], 2)
[[-1.0, -2.0]]
